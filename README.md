# Portfolio Website

## Description
A personal portfolio built with React to showcase my projects and skills.

## Features
- Responsive design
- Project showcase
- Contact form with validation
- Animations

## Technologies
- React
- React Router
- CSS / Tailwind
- Framer Motion

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open http://localhost:3000
